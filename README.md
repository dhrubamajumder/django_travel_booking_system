# ✈️ Travel Booking System

A modern Travel Booking System built with Django and Django REST Framework.  
Users can search destinations, select travel locations, book trips by date, and share reviews & ratings.

---

# 🚀 Features

## 👤 User Authentication
- User Registration
- Login / Logout
- User Profile Management

---

# 🌍 Destination Features

- Browse Travel Destinations
- Search Destinations
- View Destination Details
- Destination Categories
- Featured Destinations

---

# 📅 Booking System

- Date-wise Booking
- Travel Schedule Selection
- Booking Confirmation
- Booking History
- Booking Cancellation

---

# 🔍 Search System

Users can:
- Search by Destination Name
- Search by Location
- Search by Category

---

# ⭐ Client Reviews & Ratings

- Add Reviews
- Give Ratings
- View Client Feedback
- Review Moderation

---

# 👤 User Dashboard

Users can:
- View Profile
- Manage Bookings
- View Booking History
- Manage Reviews

---

# 📊 Reports & Analytics

- Total Bookings
- Monthly Booking Reports
- Popular Destinations
- User Statistics

---

# 🧱 Tech Stack
